Como obter as chaves
  1. Acesse trello.com/power-ups/admin
  2. Clique em New Power-Up → dê um nome → Create
  3. No menu lateral clique em API Key e copie a chave
  4. Clique em Generate a Token, autorize e copie o token
  5. Cole as duas chaves acima e clique em Salvar e conectar
